## Release 1.1

### Highlights
- Launched the end-to-end research workflow powered by multi-stage AI orchestration (Qwen-flash and Qwen3-max).
- Added comprehensive content extraction support for YouTube, Bilibili, Reddit, Tieba, and generic articles with caching to avoid duplicate fetches.
- Introduced a responsive React frontend with live WebSocket updates and session history management.
- Documented full setup paths for backend (FastAPI) and frontend (Vite + Tailwind) including config examples and troubleshooting guides.
- Delivered CLI tooling to run scrapers, batch research agents, and restore paused sessions.

### Fixes & Improvements
- Hardened browser automation flows with Playwright install guidance and debug scripts.
- Clarified configuration for Qwen API credentials, OSS transcription, and scraper tuning options.
- Expanded testing coverage with targeted scraper tests and an end-to-end workflow integration suite.
