## 1.1 版本发布

### 亮点
- 推出由多阶段 AI 编排（Qwen-flash 和 Qwen3-max）驱动的端到端研究工作流。
- 新增对 YouTube、Bilibili、Reddit、Tieba 及通用文章的全面内容抽取支持，并通过缓存避免重复拉取。
- 上线响应式 React 前端，提供实时 WebSocket 更新与会话历史管理。
- 完善后端（FastAPI）与前端（Vite + Tailwind）的完整部署文档，涵盖配置示例与故障排查指南。
- 交付命令行工具，用于运行爬虫、批量研究代理以及恢复暂停的会话。

### 修复与改进
- 加固基于 Playwright 的浏览器自动化流程，并补充安装指导及调试脚本。
- 明确 Qwen API 凭证、OSS 转写及爬虫调优选项的配置指引。
- 扩大全面测试覆盖，新增针对爬虫的重点测试和端到端工作流集成套件。
