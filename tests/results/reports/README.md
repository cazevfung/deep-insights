# Research Reports Output Directory

This directory contains final research reports generated by the Deep Research Agent tests.

## File Naming Convention

Reports are saved with the following naming pattern:
```
report_{batch_id}_{session_id}.md
```

Example: `report_251029_150500_20251029_181530.md`

## Test Execution

To run the full research agent test:

```bash
# Set API key
export DASHSCOPE_API_KEY=your_key_here  # Linux/Mac
set DASHSCOPE_API_KEY=your_key_here     # Windows

# Run test
python -m pytest tests/test_research_agent_full.py -v
# or
python tests/test_research_agent_full.py
```

## Report Structure

Each report contains:
- Research goal (selected during Phase 1)
- Generation timestamp
- Batch ID (source data identifier)
- Full research article content (generated in Phase 4)
